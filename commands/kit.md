# /kit — Agent Direct Invocation

## Actions

### `/kit explore <target>`
Explorer 에이전트로 코드베이스를 탐색합니다.

### `/kit plan <task>`
Planner 에이전트로 작업 계획을 수립합니다.

### `/kit execute <task>`
Executor 에이전트로 코드를 구현합니다.

### `/kit review [scope]`
Reviewer 에이전트로 코드를 검토합니다.

### `/kit verify [feature]`
Verifier 에이전트로 완료를 증명합니다.
