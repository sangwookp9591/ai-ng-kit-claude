# /wizard — Guided Magic Mode

비개발자도 소프트웨어를 만들 수 있도록 단계별로 안내합니다.

## How it works
1. Wizard 에이전트가 간단한 질문을 합니다
2. 답변을 바탕으로 자동으로 계획을 세웁니다
3. 한 단계씩 만들어가며 진행 상황을 보여줍니다
4. 완성되면 사용법을 알려줍니다

## Example
```
User: /wizard
Wizard: 어떤 것을 만들고 싶으세요? 🪄
User: 블로그 만들어줘
Wizard: 좋아요! 블로그를 만들어볼게요. 먼저 몇 가지 물어볼게요...
```
